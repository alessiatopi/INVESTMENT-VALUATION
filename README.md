# INVESTMENT-VALUATION
THE DATASET: The available data refer to the economic indicators of 2,482 unique companies operating in various sectors. 

THE AIM: Investigate the profitability of an investment by an Italian entrepreneur in the U.S. media sector by estimating the potential earnings per share (EPS) and considering the analysis of several economic performance indicators.

METODOLOGY USED: Regression model (Weighted Least Square).
